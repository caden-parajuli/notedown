# Notedown

Notedown is a program that makes your notes pretty. All you have to do is take notes in [Markdown](https://www.markdownguide.org), and run `notedown` on your notes folder.

## Features

- MathJax support
- Syntax highlighting
- File reloading
- Static HTML output
